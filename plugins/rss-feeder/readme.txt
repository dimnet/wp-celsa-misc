Create custom RSS feeds for your blog with permalinking and import RSS feeds.

== Description ==

Create custom RSS feeds with their own permalink and track them with a Google Analytics account, add external feeds to your blog header, import feeds into a custom feed or as posts.

Please report any bugs, if you don't tell us it how can we fix it? Please include your Wordpress version and other plugins you have installed.

== Installation ==

1. Upload the `rss-feeder` directory to your `/wp-content/plugins/` directory.
1. Activate the plugin through the `Plugins` menu in WordPress.
1. Navigate to new `RSS Feeder` main menu items.
1. Setup your permalink structure and Google Analytics.
1. For more help check the help page menu, there is also help throughout the plugin.
1. Save the changes.
